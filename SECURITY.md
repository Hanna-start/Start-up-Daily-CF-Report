# Security

- 입력은 로컬 CSV만 허용하며 원본을 외부 서비스로 전송하지 않습니다.
- 금액 계산과 예측은 로컬 Python 코드가 수행합니다.
- Gemini를 사용하는 ADK 브리핑에는 계산 결과만 전달합니다. 실제 회사 데이터 사용은 금지합니다.
- `.env`, 승인 기록, 생성 결과, Excel 파일은 Git에서 제외합니다.
- API 키를 코드·CSV·화면 캡처·영상에 넣지 마세요.
- 저장소 공개 전 `git grep -n -E "(API_KEY|AIza|password|secret)"`로 비밀정보를 점검하세요.

보안 문제는 공개 이슈 대신 저장소 소유자에게 비공개로 알립니다.
